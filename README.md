헬스장에서 사용할 수 있는 트레이너, 회원 전용 프로그램입니다.  <br>

## 개발 환경
언어 : Java<br>
툴 : Eclipse<br>
DB : mysql

## 기능
1. 로그인/회원가입
2. 트레이너 - 전체 회원 관리
3. 회원 - 출석체크, 개인 정보 관리
4. BMI, BMR 계산
